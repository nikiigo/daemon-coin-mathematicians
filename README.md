# Coordinated Stopping Strategy (Daemon Coin Game)

Monte Carlo simulation of a coordination problem.

---

## Overview

Two players observe different halves of a random sequence and must coordinate choices without communication.

---

## Model

- One sees even indices
- One sees odd indices

Each picks unknown index.

---

## Strategy

Pattern-based stopping rule using regex.

---

## Parameters

trials = 10000
serie_length = 1000

---

## Output

Survival probability.

---

## Run

python daemon_coin.py
